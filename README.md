# 📚 Bookstore App

A full-stack web application to browse and manage books, built using **MERN stack (MongoDB, Express, React, Node.js)**. Deployed with a monorepo setup on [Render](https://render.com).

## 🌐 Live Demo

- **Frontend**: [https://bookstoreapp-yd4w.onrender.com](https://bookstoreapp-yd4w.onrender.com)
- **Backend**: [https://bookstoreapp-q3mj.onrender.com](https://bookstoreapp-q3mj.onrender.com)

---

## 🏗️ Tech Stack

### Frontend:
- React
- React Router
- Axios
- Vite

### Backend:
- Node.js
- Express.js
- MongoDB (Atlas)
- Mongoose
- JWT Authentication
